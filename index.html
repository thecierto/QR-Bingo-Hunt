📡 Tracking Pixel / Retargeting

Paste your Meta Pixel, Google Tag, TikTok Pixel, or any custom HTML/script here. It fires on both the Winner and Losing pages so you can retarget all participants after the event. Use {{EVENT}} as a placeholder — replaced with winner or miss at runtime.

Common platforms:
Meta: Paste your base <script>fbq(...)</script>, add fbq('track','{{EVENT}}')
Google Ads: Use your gtag conversion snippet, swap {{EVENT}} for the event name
TikTok: Paste base pixel code and use ttq.track('{{EVENT}}')
Custom: Any valid HTML/JS — including a 1×1 <img> beacon
